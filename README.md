# Secure Authentication System

A secure authentication system built using Java, Spring Boot, Spring Security, JWT, OTP verification, MySQL, and frontend pages.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- OTP Verification
- MySQL
- REST APIs
- Maven
- HTML
- CSS
- JavaScript

## Features
- User registration
- User login
- JWT-based authentication
- OTP generation and verification
- Password encryption
- Secured API endpoints
- Frontend authentication pages

## How to Run
1. Clone the repository
2. Configure MySQL database in `src/main/resources/application.properties`
3. Run the Spring Boot application
4. Open the frontend pages from the `frontend` folder
5. Test APIs using Postman or browser
